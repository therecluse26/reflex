# Scoop Bucket for Reflex

This is the Scoop bucket for [Reflex](https://github.com/reflex-search/reflex), a local-first, structure-aware code search engine for AI agents.

## Usage

```powershell
scoop bucket add reflex https://github.com/reflex-search/scoop-reflex
scoop install reflex
```

## Auto-Update

This bucket uses the Excavator GitHub Action to automatically check for new releases every 6 hours and update the manifest accordingly.
